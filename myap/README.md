# Argumento

Argumento is an educational web application inspired by *Papers, Please* and *Duolingo*. The platform trains users to distinguish legitimate news from misinformation through interactive gameplay and decision-based analysis.

## Project Overview

- **Objective:** Educate users about misinformation, improve critical thinking, and raise awareness of the societal impact of fake news.
- **SDG Alignment:** United Nations Sustainable Development Goal 4: Quality Education.
- **Primary Platform:** Flutter Web (mobile-ready for optional scalability).

## Core Features

- Authentication: Sign up, login, logout
- Dashboard: user progress overview, daily challenges, missions
- News analysis: submit or view news content and decide whether it is real or fake
- Result feedback: correctness indicator, explanation, and learning feedback
- Campaign system: join anti-hoax campaigns and view active campaigns
- Gamification: shop, rewards, XP, coins, badges
- Leaderboard: view rankings among users
- Profile system: view own profile and other user profiles

## UI/UX Direction

- Clean and modern design with gamified visuals
- Simple decision-based interaction inspired by *Papers, Please*
- Friendly for university students and general audiences
- Bright, engaging style without being overly childish

## Getting Started

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Open the project in your IDE
3. Run:
   ```bash
   flutter pub get
   flutter run -d chrome
   ```

## Notes

This project focuses on frontend development using Flutter and is structured for web deployment with mobile-ready scalability.
